PROJECT DOCUMENT – HOUSING LOTTERY APPLICATION SYSTEM
1. Project Overview
This project aims to build an online Housing Lottery & Plot Allocation Management System for
government housing schemes. The system will simplify plot applications, automate verification,
handle payments, manage applicant categories, and process refunds and allotments.
2. Purpose of the Project
The system provides citizens with an easy, transparent, and secure method to submit housing
scheme applications online. It removes manual paperwork, reduces errors, ensures fairness in
lottery allotments, and enables authorities to manage records digitally.
3. Who This Project Is For
• Government Municipal Bodies
• Housing Development Authorities
• Citizens applying for plots
• Officers responsible for verification and approval
• Administrators who manage schemes and reports
4. Key Modules & Pages Required
A. Applicant Portal
• Home page with scheme details
• Eligibility checker
• Online application form
• Document upload section
• Payment gateway integration
• Application status tracking
• Refund status page
• Allotment result page
B. Admin Portal
• Dashboard (applications count, category-wise data)
• Manage schemes
• View & verify applications
• Upload lottery results
• Manage refunds and reports
• Role-based access management
5. What the Project Will Do
• Allow users to fill online applications
• Collect application fees securely
• Verify eligibility and income category
• Auto-segregate applicants into categories (EWS, LIG, MIG, HIG)
• Conduct lottery and publish results
• Generate allotment letters
• Process refunds for cancelled/ineligible applications
6. System Workflow
1. User opens portal → views scheme details
2. Creates account → verifies mobile (OTP)
3. Fills application form → uploads documents
4. Pays registration fees
5. System validates eligibility
6. Verified applications go to lottery pool
7. Lottery executed → results published
8. Selected users get allotment letter
9. Others get refund as per rules
7. Detailed Description
• Multi-role login system for applicants & admin
• Auto income category calculations
• Payment logs and auto-refund generation
• PDF allotment letter creation
• Document verification workflow
• Secure cloud-based architecture
• Full audit logs for legal compliance
• Multilingual support (English/Hindi)
